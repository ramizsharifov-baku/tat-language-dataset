# Tat Language Dataset
Compiled and provided by Ramiz Sharif (Azerbaijan)

## Project Description
This repository contains linguistic materials related to the Tat language, including alphabet, vocabulary, example sentences, texts, and basic grammar notes.

This project was initiated and compiled by Ramiz Sharif as part of an effort to preserve, digitize, and document the Tat language and to make Tat language materials available for research, education, and future use in language technologies and Artificial Intelligence systems.

## Author
Ramiz Sharifov
Azerbaijan
Project start: 2024
Digital publication: 2026 (GitHub)

## Project Goals
- Preservation of the Tat language
- Creation of a digital Tat language dataset
- Support for linguistic research
- Support for AI and language technology development
- Documentation of alphabet, vocabulary, and grammar

## Repository Structure
- Alphabet.txt – Tat alphabet
- Words.txt – Tat vocabulary
- Sentences.txt – Example sentences
- Texts.txt – Sample texts
- Grammar.txt – Basic grammar notes
- About.txt – Project and authorship information
